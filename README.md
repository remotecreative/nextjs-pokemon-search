# Pokemon Search

A simple Pokemon search app built with Next.js and TypeScript. Search for any Pokemon and view its details including types, stats, height, and weight.

## Demo

https://github.com/user-attachments/assets/68011e93-7ed5-4ecd-8dc8-4cd106faa645

## Features

- Search form available on every page
- Server-side rendered Pokemon pages
- Displays Pokemon image, types (with color-coded badges), physical stats, and base stats
- Loading and error state handling

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Usage

Enter a Pokemon name in the search bar and press Search. You'll be taken to that Pokemon's page with all its details.

Example Pokemon to try:
- pikachu
- charizard
- bulbasaur
- mewtwo
- gengar
- eevee
- snorlax
- dragonite

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- PokeAPI
